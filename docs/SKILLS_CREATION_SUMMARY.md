## スキルの特徴

### 1. 体系的なフロー

各スキルは複数の Phase に分かれており、段階的に問題を解決：
- Phase 1: 情報収集
- Phase 2-N: 分析・対応
- 最終 Phase: 結果の記録・再発防止

### 2. チェックリスト形式

各 Phase に具体的なチェックリストを含む：
- 実施すべき項目が明確
- 漏れを防止
- 進捗を可視化

### 3. 実践的な例

各スキルに実装例・修正例を含む：
- ❌ 不正な例
- ✅ 正しい例
- 修正手順

### 4. トラブルシューティング

各スキルに一般的な問題と解決方法を記載：
- 問題が発生した場合の対処方法
- 参考資料へのリンク

## スキル使用方法

### 方法 1: スキルファイルを直接開く

```bash
# スキルファイルを開く
code ~/.kiro/skills/e2e-test-fix-flow.md

# 手順に従って実行
```

### 方法 2: Kiro IDE で読み込む

1. Kiro IDE を開く
2. コマンドパレット（Ctrl+Shift+P）を開く
3. `Kiro: Activate Skill` を実行
4. スキル名を選択

### 方法 3: チャットで参照

Kiro チャットで質問：
```
E2E テストが失敗しました。修正方法を教えてください。
```

Kiro が自動的にスキルを参照して対応します。
